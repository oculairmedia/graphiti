⚠️ About the Authentication Error

The FalkorDB Browser UI at http://localhost:3100 expects to connect to a Redis instance. Since we're running FalkorDB in a Docker container with mapped ports, the UI might have trouble connecting to the backend.

How to Access FalkorDB:

Option 1: Use Redis CLI (Recommended)

# Connect to FalkorDB
docker exec -it graphiti-falkordb-1 redis-cli

# Select your graph
GRAPH.QUERY graphiti_migration "MATCH (n) RETURN count(n)"

# Example queries:
GRAPH.QUERY graphiti_migration "MATCH (n) RETURN n.name, labels(n) LIMIT 10"
GRAPH.QUERY graphiti_migration "MATCH (a)-[r]->(b) RETURN a.name, type(r), b.name LIMIT 10"

Option 2: Use Python Scripts

# Run the migration
python3 migrate_working.py

# Test the connection
python3 test_new_port.py

Option 3: Use RedisInsight

For a better GUI experience, you can use RedisInsight:

  1. Download RedisInsight from redis.com
  2. Connect to localhost:6389
  3. Use the Graph module to query your FalkorDB graphs
